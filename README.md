# Amazon Sales & Revenue Analysis Dashboard

## 📊 Project Overview
This project was developed during my Data Analytics Internship to analyze historical sales and revenue data from 2016 to 2026. The goal of this interactive dashboard is to provide executive leadership with actionable insights regarding revenue trends, profitability margins, and product performance.

## 🚀 Key Features
- **Data Transformation (ETL):** Cleaned and structured raw transactional data using Power Query.
- **Dynamic KPI Tracking:** Built optimized DAX measures to isolate Total Revenue, Total Orders, Average Order Value (AOV), and Net Profit Margin.
- **Interactive Visualizations:** Implemented continuous dual-axis trend analysis, segment contribution breakdown, and dynamic Top N product filtering.
- **Executive Slicers:** Enabled immediate multi-dimensional filtering across macro Regions and Product Categories.

## 🛠️ Tech Stack
- **Data Source:** Microsoft Excel (`.xlsx`)
- **Analytics & BI Platform:** Microsoft Power BI Desktop
- **Data Modeling Language:** DAX (Data Analysis Expressions)

## 💡 Key Business Insights Generated
1. **Product Concentration:** A small cohort of top-tier products drives a substantial share of gross revenue, highlighting a critical area for supply chain and inventory optimization.
2. **Segment Performance:** The Consumer segment yields the highest overall transaction volume, representing the primary driver of core market footprint.
3. **Margin Health:** The tight calibration between the Revenue trend and Net Profit Margin % indicates sustainable baseline pricing across historical promotional cycles.
